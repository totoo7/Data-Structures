\doxysubsection{Detailed Description}
A class that represents a \doxylink{classTrie}{Trie} (prefix tree) data structure. 

This class provides operations for inserting, removing, checking if a word exists, printing all words in the \doxylink{classTrie}{Trie}, and providing autocomplete suggestions.
