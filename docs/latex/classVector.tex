\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Vector$<$ T $>$}
A dynamic array-\/like container that provides random access to elements. 

This class implements a custom vector which can store elements and automatically resize when it reaches its capacity. It also supports various container operations such as adding elements, removing elements, and accessing the elements using iterators or indices.
