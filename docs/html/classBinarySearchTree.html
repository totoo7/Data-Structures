Detailed Description

template<typename T>
class BinarySearchTree< T >

A template-based Binary Search Tree implementation.

Provides basic operations such as insertion, deletion, search, height calculation, and in-order traversal printing.

Template Parameters
TThe type of the elements stored in the Binary Search Tree. Must support comparison operators.
